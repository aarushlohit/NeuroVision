Key Components Explained

Residual Blocks (Skip Connections)

Allow gradients to flow directly through the network. Each block learns residual functions F(x) = H(x) - x, enabling 50+ layer depth.

Batch Normalization

Normalizes activations in each mini-batch, reducing internal covariate shift and allowing higher learning rates.

Transfer Learning

Pre-trained on ImageNet (1.2M images, 1000 classes) then fine-tuned on medical imagery for robust feature extraction.

Dropout Regularization

30% dropout in FC layers randomly deactivates neurons during training to prevent overfitting.
